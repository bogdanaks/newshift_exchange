# Newshift Example Exchange

## Setup

1. Copy .env from .env.example parameters in next reps:

```
server
```

2. Create local database newshift and start on port 5432

3. Start apps:

```
front - npm run start
server - npm run dev
```

4. Test

```
front - http://localhost:3000
server - http://localhost:5000/api/v1/...

```
